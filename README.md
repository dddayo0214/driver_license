# 自動化駕照報名工具（Auto License Registration Bot）

一個以 **Python + Selenium + UV**
建立的自動化駕照報名程式，可自動填寫學員資料、自動選擇考試日期並送出報名表單。\
只需提前輸入個人資料，即可一鍵完成流程。

------------------------------------------------------------------------

## ✨ 功能特色（Features）

-   🔄 **全自動化流程**：自動進入系統 → 填資料 → 選日期 → 提交
-   🧾 **支援事先設定資料**（姓名、身分證、生日、電話、班別等
-   ⚡ **使用 `uv` 管理環境**（快速、乾淨、可重現
-   🌐 **Selenium 自動化瀏覽器操作**
-   🧩 **模組化架構，易於維護與擴充**

## 🛠 環境需求（Requirements）

-   Python **3.9+**
-   Chrome 瀏覽器
-   uv（快速 Python 套件管理工具）
-   Windows / macOS / Linux 皆可使用

安裝 UV：

    pip install uv

------------------------------------------------------------------------

## 📦 安裝（Installation）

1.  Clone 專案

    git clone <https://github.com/dddayo0214/driver_license.git>
    cd driver-license

2.  使用 uv 安裝依賴

    uv sync

## ▶️ 使用方式（Usage）

使用 uv 執行主程式：

    uv run src/main.py

輸入資料後點擊開始按鈕
程式會：

1.  啟動 Chrome
2.  自動前往駕照報名網站
3.  自動填入資料
4.  自動選擇日期
5.  自動送出表單
6.  回傳結果

## ❓ 常見問題（FAQ）

### 1. ChromeDriver 版本不符？

你使用的是 webdriver-manager → 會自動偵測與下載正確版本。
若遇到錯誤可更新：

    uv pip install --upgrade webdriver-manager

------------------------------------------------------------------------

### 2. 報名網站改版怎麼辦？

更新表單欄位 selector / XPath 即可。
如需要，我可協助你調整。
